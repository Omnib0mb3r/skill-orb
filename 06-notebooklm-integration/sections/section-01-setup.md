## Test Fixtures

Create the following fixture files in `tests/fixtures/`. These are shared across multiple test suites — creating them now ensures sections 03, 04, 06, and 08 can all reference them without re-creating.

### `tests/fixtures/sample-session.jsonl`

A realistic multi-line JSONL log file representing a DevNeural session. Each line is one `LogEntry` JSON object. This fixture must include:

- At least 6 entries
- Two different `project` values (so primary_project detection can be tested by frequency)
- All four `connection_type` values: `project->tool`, `project->skill`, `project->project`, `tool->skill`
- Timestamps spaced apart (so session_start / session_end can be verified)
- At least two different `tool_name` values
- At least one entry with `tool_input.file_path` and one with `tool_input.path` (for the generator's file-path extraction)

Example shape for one entry (fields match 01-data-layer's LogEntry):

```json
{"timestamp":"2026-03-30T10:00:00Z","project":"github.com/Omnib0mb3r/DevNeural","source_node":"project:github.com/Omnib0mb3r/DevNeural","target_node":"tool:Read","connection_type":"project->tool","tool_name":"Read","stage":"post","tags":[],"tool_input":{"file_path":"/src/types.ts"}}
```

Design the fixture so tests can assert deterministic values (exact project names, exact timestamps, exact connection counts).

### `tests/fixtures/sample-weights.json`

A minimal `weights.json` fixture representing graph edges. The graph-reader tests (section 04) will use this file as the API fallback. Include:

- At least 4 edges for the primary project from the JSONL fixture
- At least 1 edge where `first_seen` matches `2026-03-30` (for `new_connection` insight detection)
- At least 1 edge where `last_seen` matches `2026-03-30` AND `raw_count` is one of `[10, 25, 50, 100]` (for `weight_milestone` heuristic)
- At least 3 edges for the same project so the `high_weight` top-3 logic can be exercised
- Node IDs stored with `project:` prefix (e.g., `"project:github.com/Omnib0mb3r/DevNeural"`)

Example shape:

```json
{
  "edges": [
    {
      "source_node": "project:github.com/Omnib0mb3r/DevNeural",
      "target_node": "tool:Read",
      "weight": 0.95,
      "raw_count": 50,
      "first_seen": "2026-03-01",
      "last_seen": "2026-03-30"
    }
  ]
}
```
